# Customer Segmentation with K-Means Clustering

## Project Overview
This project applies K-Means clustering to segment customers based on their purchase behavior for CodeClause Internship.

## Project Details
- **Project ID**: #CC69853
- **Domain**: Data Science Intern
- **Level**: Entry Level
- **Technologies Used**: Python, Pandas, Scikit-learn, Matplotlib

## Dataset
The Mall Customers dataset containing:
- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

## Methodology
1. Data Preprocessing and Exploration
2. Feature Selection (Annual Income & Spending Score)
3. Data Standardization
4. Elbow Method for Optimal Cluster Determination
5. K-Means Clustering Implementation
6. Cluster Visualization and Analysis

## Results
- Identified 5 distinct customer segments
- Visualized clusters with centroid markers
- Analyzed characteristics of each customer segment

## How to Run
1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Open and run `customer_segmentation.ipynb` in Jupyter Notebook or Google Colab
